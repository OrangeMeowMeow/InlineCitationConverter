How to Use

  1. Prepare your three files:
    • Reference.txt - Text version of references
    • Main.tex - Your main LaTeX document
    • Reference.bib - BibTeX file with citation keys
  2. Upload all three files using the form
  3. Click "Convert Citations" to process
  4. Review any conversion messages
  5. Download the converted LaTeX file

Requirements

  • Ensure your citations are in APA format: (Author, Year)
  • Multiple citations should be separated by semicolons
  • Textual citations should be in "Author (Year)" format

Example

Input: Two plus two equals five (Li, 1989).
Output: Two plus two equals five \citep{Li_1989_}.
Input: Li (1989) demonstrated that...
Output: \citet{Li_1989_} demonstrated that...
